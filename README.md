# 作用
进行多个点赞, 五个帖子浏览, 之后领取每日奖励
仅使用过繁中测试, 其他语言不支持
# 用法
- Git clone项目后, pip install -r requirement安装依赖
- 配置setting.py文件的账户密码, 区服只有日本/韓國/北美/東南亞/全球
- py main.py执行
